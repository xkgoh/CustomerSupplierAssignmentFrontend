# CustomerSupplierAssignment

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 13.0.4.

## Code Organization
The project is organized into several components:
1. Product
2. Product Sales Report
3. Customer
4. Monthly Sales Report
5. Models (where all the data models are stored) 
6. Credentials (For the username/password input field to be used across all views)

## To run
1. Execute `npm install`
2. Run `ng serve`
3. Navigate to http://localhost:4200
4. Enter username and password (admin:admin or reader:reader) before executing any API calls
